## 核心摘要

本文详细介绍了 GitHub Fine-grained personal access tokens（细粒度个人访问令牌）的权限体系。与传统的经典令牌不同，细粒度令牌允许对每个权限进行单独控制（无访问、只读、读写），涵盖仓库核心内容、安全警报、协作流程、高级配置、组织管理、包管理等多个维度。文章强调了最小权限原则的重要性，建议根据实际需求精确配置权限，以降低令牌泄露带来的安全风险。
