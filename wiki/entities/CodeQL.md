## 定义

CodeQL 是 GitHub 提供的代码分析引擎，用于自动检测代码中的安全漏洞和错误，支持多种编程语言的静态代码分析。

## 关键信息

- GitHub 原生代码扫描工具
- 生成 Code scanning alerts（代码扫描警报）
- 可以发现代码中的漏洞和错误
- 需要 Code scanning alerts 权限来管理警报
