## 定义

GitHub Codespaces 是 GitHub 提供的云端开发环境服务，允许开发者在浏览器中直接编写、运行和调试代码，无需本地配置开发环境。

## 关键信息

- 提供完整的云端 IDE 体验
- 支持生命周期管理（启动、停止、删除）
- 可以配置 Codespaces secrets（环境变量等敏感信息）
- 需要多个权限：Codespaces、Codespaces lifecycle admin、Codespaces metadata、Codespaces secrets
