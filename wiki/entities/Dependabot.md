## 定义

Dependabot 是 GitHub 提供的自动化依赖管理工具，能够自动检测项目依赖中的安全漏洞，并创建 Pull Request 来更新有漏洞的依赖项。

## 关键信息

- 自动检测依赖项中的安全漏洞
- 生成 Dependabot alerts（依赖项警报）
- 可以管理 Dependabot secrets（用于访问私有依赖库的敏感信息）
- 需要特定权限：Dependabot alerts、Dependabot secrets
