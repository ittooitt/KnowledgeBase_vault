## 定义

GitHub Packages 是 GitHub 提供的包管理服务，允许开发者在 GitHub 上托管和管理软件包，支持多种包管理器（如 npm、Maven、NuGet、Docker 等）。

## 关键信息

- 支持上传、下载和删除包
- 与 GitHub 仓库紧密集成
- 需要 Packages 权限来管理包的访问
- 可以设置包的可见性（公开或私有）
