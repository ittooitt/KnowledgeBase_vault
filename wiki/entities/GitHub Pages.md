## 定义

GitHub Pages 是 GitHub 提供的静态网站托管服务，允许用户直接从 GitHub 仓库发布静态网站，常用于项目文档、个人博客、项目展示页面等。

## 关键信息

- 支持 Jekyll 静态站点生成器
- 可以通过 GitHub Actions 自动构建和部署
- 需要 Pages 权限来配置和构建站点
- 通常与 CI/CD 流程结合使用
