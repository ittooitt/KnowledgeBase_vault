## 定义

GitHub 是全球最大的代码托管平台和开发者协作平台，基于 Git 版本控制系统，提供代码仓库管理、协作开发、CI/CD、安全扫描、包管理等全方位的软件开发服务。

## 关键信息

- 提供 Fine-grained personal access tokens（细粒度个人访问令牌），允许对每个权限进行精确控制
- 核心功能包括：代码托管、Issues 追踪、Pull Requests、GitHub Actions、GitHub Pages、GitHub Packages 等
- 支持多层级的权限管理：仓库级、组织级、个人级
