## 定义

软件供应链安全是指保护软件开发、构建、分发全流程中的安全性，包括源代码、依赖项、构建工具、部署流程等各个环节，防止恶意代码注入、依赖漏洞利用等安全威胁。

## 关键信息

### GitHub 中的相关功能

- **Dependabot**：自动检测依赖项漏洞
- **CodeQL**：代码扫描，发现安全漏洞
- **Secret scanning**：扫描代码中的敏感信息（密码、API 密钥等）
- **Attestations**：验证工件的来源和完整性

### 权限管理

需要特定权限来管理这些安全功能：
- Code scanning alerts
- Dependabot alerts
- Secret scanning alerts
- Repository security advisories

### 重要性

- 防止供应链攻击
- 保护代码和数据安全
- 确保软件交付的可信性
